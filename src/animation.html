💡 請將滑鼠移到文字上體驗動畫效果

動畫演示

你好，我是 MUKI，很高興認識你，Nice to meet you.

技術說明

這個動畫使用了 Web Animations API 來實現文字打散效果。當滑鼠懸停在文字上時，每個字符會隨機移動到不同位置並旋轉，移開時會回到原位。

主要技術包括：字符分割、隨機位置計算、CSS 動畫、以及 Web Animations API 的 animate() 方法。
