Web Notifications API 搭配 Service Workers 發送通知

完整文章：Web Notifications API 搭配 Service Workers 發送通知

請先點擊「請求通知權限」，才能開始此功能

使用方法

1. 打開瀏覽器開發者工具，切換到 Application 分頁，點擊 Service Workers，可以看到 Service Workers 的狀態

2. 點擊「Push」，可以看到 Push 的註冊狀態

3. 在 Push 對話框填入以下資料，再按下 Push 就會發送通知

{
  "title": "這是通知標題",
  "body": "這是通知內容",
  "data": {
    "url": "https://muki.tw/ngrok-err-ngrok-6024/"
  }
}
